# 实践操作目录

这个目录放能直接服务训练和比赛的上机文档。阅读顺序建议从“跑通导航”开始，再学模型、仿真、传感器，最后进入调参。

推荐顺序：

1. `目的对应操作指南.md`
2. `Turtlebot3仿真导航快速跑通.md`
3. `SLAM边建图边导航.md`
4. `机器人模型URDF与RViz可视化.md`
5. `Gazebo仿真与EKF里程计.md`
6. `传感器建图定位与代价地图.md`
7. `Nav2代价地图参数配置.md`
8. `机器人Footprint轮廓配置.md`
9. `Nav2规划器与控制器插件配置.md`
10. `Nav2调参与故障诊断手册.md`
11. `实践提交与截图清单.md`

遇到问题时：

- 想知道 `目的.md` 四条要求分别怎么做：看 `目的对应操作指南.md`。
- 不知道怎么启动完整导航：看 `Turtlebot3仿真导航快速跑通.md`。
- 没有地图，需要边建图边导航：看 `SLAM边建图边导航.md`。
- RViz 模型、TF、雷达不正常：看模型、Gazebo、传感器三篇。
- 路径贴墙、过不了门、到点转圈：看调参诊断手册。
- 需要交答题报告：看实践提交与截图清单。

常见启动错误：

- `/spawn_entity` 服务不可用、`gzserver exit code 255`：看 `../错误排查/Gazebo_spawn_entity服务不可用与gzserver退出255.md`。
